Surveyor App

#Back-End
1. Run Server with
```
$ nodemon
```

#Front-End
1. Run Client with
```
$ npm run start
```

#Features
## Admin
- Can create a new User(Admin/Surveyor)
- Can Approve or Decline a Product
- Can see approved product

## Surveyor
- Can add a new product 
- Can take down declined product 
- Can see approved product

## Visitor
- Can see approved product
- Can't Login
